pub struct MutablePrimitiveArray<T: NativeType> { /* private fields */ }
Expand description

The Arrow’s equivalent to Vec<Option<T>> where T is byte-size (e.g. i32). Converting a MutablePrimitiveArray into a PrimitiveArray is O(1).
