Expand description

The Arrow‚Äôs equivalent to Vec<Option<bool>>, but with 1/16 of its size. Converting a MutableBooleanArray into a BooleanArray is O(1).

Implementation

This struct does not allocate a validity until one is required (i.e. push a null to it).
